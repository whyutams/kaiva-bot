# 🤖 Kaiva - Discord Bot - Source Code
Bot Discord multifungsi dengan fitur utilitas dan hiburan. Dibuat dengan `discord.js v14+` dan `Node.js`. Juga dilengkapi dengan web server menggunakan Express.js.
